# Cards

Canonical card components (Group 4 spec):

- `KpiCard` — current KPI score + scholarship status
- `AnalyticsCard` — small analytics widget for dashboard
- `StudentCard` — student summary card
- `PenaltyCard` — red-themed penalty card with severity badge

Each card should:
- accept loading skeleton state
- render explainable hover/tooltip (Group 6)
- use design tokens, never hard-coded colors
