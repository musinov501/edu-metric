# Tables

Canonical table components (Group 4 spec):

- `StudentTable` — columns: Avatar, Name, Group, GPA, Score, Attendance, Scholarship Status, Actions
- `LeaderboardTable` — rank rows below the Top-3 podium
- `ApprovalTable` — pending approvals with preview + approve/reject buttons

Responsiveness rule (Group 4): tables collapse to stacked cards on mobile.
